SWIMMING SCHOOL

This page contains the case study of Swimming School Open-Source Project which includes the Project Overview, Tools Used and Live Links to the official product.

Project Overview

Swimming School is an Open-Source project which is a simple and clean website template for Institution. I created this project for Institutions to quickly build a good-looking and fast-performing website without having to code their first Website from scratch.

Swimming School is now being used by thousands of Institutions globally and it has helped many people in landing jobs & opportunities which makes me happy that my creation is helping other people in building their careers.
